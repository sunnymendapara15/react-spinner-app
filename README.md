# React Spinner App

A lightweight React starter that lets you add names or ideas and spin to choose one at random.

## Getting started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the dev server:
   ```bash
   npm run dev
   ```
3. Add items, spin the wheel, and enjoy the randomness!

## Available scripts

- `npm run dev`: Launches Vite dev server.
- `npm run build`: Produces an optimized production build.
- `npm run preview`: Serves the production build locally.
